# The "Undo" Effect on Decision Thresholds

## Status: monkey-tested

## Description
A Speed-Accuracy Trade-Off (SATO) experiment testing how an "undo" safety net changes cognitive decision thresholds. Participants classify random dot-motion direction (Left vs Right) under two conditions: Block A (choice is final) and Block B (300ms correction window available). Hypothesis: the Undo option lowers the response threshold, producing faster but less accurate initial responses.

## Plan
- **Paradigm**: Two-choice perceptual categorization (dot-motion direction)
- **Stimuli**: Pre-generated static dot-pattern PNGs at 3 coherence levels (80%/50%/30%)
- **Key manipulation**: Block A = final choice; Block B = 300ms post-response correction window
- **Undo mechanic**: Two-trial chain — initial response trial branches to a correction trial
- **Counterbalancing**: Two `<expt>` elements (AB vs BA block order)
- **Modular architecture**: 8 include files with `ut_` namespace prefix
